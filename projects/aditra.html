Aditra Project

Project Description

The Automatic Disinfecting Trashbin (ADITRA) is an innovative solution that combines hardware and software to create a safer waste disposal system.

Key Features

  • Automatic lid operation using proximity sensors and servo motors
  • UV Light disinfection system
  • Precise control using Arduino servo mechanism

Technologies Used

  • Arduino Microcontroller
  • C++ Programming
  • Ultrasonic Sensors
  • UV-C Disinfection System

Development Period

March 2022 - June 2023

Institution

Adelle Grace Montessori School
